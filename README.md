# NamaXPress

NamaXPress is a ride-sharing and driver booking web application built with **Python Flask**, **HTML/CSS**, and **Socket.IO**. It allows riders to find available drivers, book rides, and chat privately with drivers in real-time. Drivers can create bookings and manage ride availability.

---

## **Features**

### For Riders:
- Browse available drivers for today.
- View driver pickup and dropoff locations.
- Check available seats and timing.
- Private chat with drivers in real-time (mobile-friendly chat sidebar).
- Easy navigation with a responsive design for both desktop and mobile.

### For Drivers:
- Create new bookings for rides.
- See a list of rider requests.
- Manage available seats and ride status.
- Delete or update bookings as needed.
- Receive messages from riders directly via the chat sidebar.

### General:
- Mobile-first responsive design.
- Interactive chat powered by **Socket.IO**.
- Stylish UI with gradients and smooth animations.
- Easy to navigate back buttons and user-friendly interface.
